# React Router & API Integration

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais do React relacionados à navegação entre páginas, gerenciamento de estados e consumo de APIs.

A aplicação não teve foco na construção de funcionalidades complexas ou no desenvolvimento do back-end. O principal objetivo foi exercitar a implementação de rotas utilizando React Router e a integração com APIs através do Axios, simulando cenários comuns encontrados no desenvolvimento de aplicações modernas.

## 🚀 Objetivos de Aprendizado

Durante o desenvolvimento deste projeto, foram praticados os seguintes conceitos:

- Configuração e utilização do React Router para navegação entre páginas.
- Realização de requisições HTTP para consumo de APIs.
- Tratamento de respostas e possíveis erros retornados pelas APIs.
- Gerenciamento de estados utilizando os recursos do React.
- Integração do Axios para simplificar e organizar requisições HTTP.
- Configuração de interceptors do Axios para automatizar a inclusão de tokens de autenticação nas requisições.
- Organização do código seguindo boas práticas para aplicações React.

## 🔗 API Utilizada

Este projeto consome uma API REST disponibilizada separadamente e utilizada exclusivamente para fins de estudo e prática de integração entre front-end e back-end.

Para que a aplicação funcione corretamente, é necessário configurar e executar a API antes de iniciar o projeto React.

### Repositório da API

https://github.com/viniciosneves/4870--api-com-docker

Todas as instruções de instalação, configuração e execução da API estão disponíveis no README do próprio repositório.

Após iniciar a API, verifique se a URL configurada no Axios corresponde ao endereço onde a API está sendo executada.

## ▶️ Como Executar o Projeto

### Pré-requisitos

- Node.js
- npm ou yarn
- API em execução

### 1. Clonar o repositório

```bash
git clone <https://github.com/gabriel-pereira0/Projeto_Code_Connect>
```

### 2. Acessar a pasta do projeto

```bash
cd Code_Connect
```

### 3. Instalar as dependências

```bash
npm install
```

### 4. Configurar e iniciar a API

Antes de iniciar o front-end, siga as instruções disponíveis no repositório da API:

https://github.com/viniciosneves/4870--api-com-docker

### 5. Iniciar a aplicação React

```bash
npm start
```

ou

```bash
npm run dev
```

Dependendo da configuração utilizada no projeto.
